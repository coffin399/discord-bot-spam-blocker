主な機能

Discordサーバーを安全に保つための包括的なスパム対策

許可リスト制御

許可されたBOTのみが投稿できるよう制御。未許可のBOTの投稿は自動削除されます。

チャンネル監視

特定のチャンネルのみを監視対象に設定可能。必要な場所だけを保護できます。

管理者権限

管理者権限を持つユーザーは常に投稿可能。権限のあるユーザーを制限しません。

動的設定

コマンドによる動的な設定変更。サーバーを再起動することなく設定を更新できます。

警告メッセージ

削除時の警告メッセージ表示（オプション）。何が削除されたかを透明性を持って表示。

YAML設定

YAMLファイルによる簡単な設定管理。直感的で分かりやすい設定ファイル。

簡単セットアップ

数分でDiscordサーバーを保護できます

1

BOTを作成

Discord Developer PortalでBOTを作成し、サーバーに招待

Discord Developer Portal → New Application → Bot → Add Bot
2

設定ファイルを編集

config.yamlファイルでBOTトークンと許可BOTを設定

bot_token: "YOUR_BOT_TOKEN_HERE"
allowed_bots:
  - "123456789012345678"
monitored_channels:
  - "345678901234567890"
3

BOTを起動

Pythonスクリプトを実行してBOTを起動

python main.py
GitHubでダウンロード コマンド一覧を見る

管理コマンド

すべてのコマンドは管理者権限が必要です（プレフィックス: !!!）

BOT管理

!!!add_bot <BOT_ID>
許可BOTリストに追加
!!!remove_bot <BOT_ID>
許可BOTリストから削除
!!!list_bots
許可されているBOTの一覧を表示

チャンネル管理

!!!add_channel <CHANNEL_ID>
監視対象チャンネルを追加

その他

!!!reload_config
設定ファイルを再読み込み
