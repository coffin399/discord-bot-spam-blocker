セットアップガイド

数分でDiscordサーバーを保護できます

前提条件

セットアップを開始する前に必要なもの

Python 3.8+

Python 3.8以上がインストールされている必要があります

python --version

Discordアカウント

Discordアカウントとサーバーの管理者権限が必要です

サーバー

保護したいDiscordサーバーの管理者権限が必要です

1

プロジェクトをダウンロード

GitHubからプロジェクトをダウンロードまたはクローンします

GitHubからダウンロード

ZIPファイルをダウンロードして解凍

ZIPをダウンロード

Gitでクローン

Gitを使用してリポジトリをクローン

git clone https://github.com/your-username/discord-bot-spam-blocker.git
2

依存関係をインストール

必要なPythonライブラリをインストールします

Windows（自動インストール）

start.batファイルを実行すると自動的に依存関係がインストールされます

start.bat

手動インストール

pipを使用して手動でインストール

pip install discord.py pyyaml

requirements.txtから

requirements.txtファイルから一括インストール

pip install -r requirements.txt
3

Discord BOTを作成

Discord Developer PortalでBOTを作成し、サーバーに招待します

3.1

アプリケーションを作成

  1. Discord Developer Portalにアクセス
  2. 「New Application」をクリック
  3. アプリケーション名を入力（例：Spam Blocker）
  4. 「Create」をクリック
3.2

BOTを作成

  1. 左サイドバーの「Bot」をクリック
  2. 「Add Bot」をクリック
  3. 「TOKEN」セクションで「Copy」をクリックしてトークンをコピー
  4. トークンを安全な場所に保存
3.3

権限を設定

  1. 「Privileged Gateway Intents」で以下を有効化：
  2. ✅ MESSAGE CONTENT INTENT
  3. ✅ SERVER MEMBERS INTENT
3.4

BOTをサーバーに招待

  1. 「OAuth2」→「URL Generator」に移動
  2. 「SCOPES」で「bot」を選択
  3. 「BOT PERMISSIONS」で以下を選択：
  4. ✅ Read Messages/View Channels
  5. ✅ Send Messages
  6. ✅ Manage Messages
  7. ✅ Read Message History
  8. 生成されたURLからサーバーにBOTを招待
4

設定ファイルを編集

config.yamlファイルでBOTトークンと設定を行います

初回起動

初回起動時にconfig.yamlが自動生成されます

python main.py

設定ファイルの編集

config.yamlファイルを開いて以下の設定を行います

基本設定

# Discord BOTトークン（必須）
bot_token: "YOUR_BOT_TOKEN_HERE"

# 許可するBOTのIDリスト
allowed_bots:
  - "123456789012345678"  # 許可したいBOTのIDを追加
  - "234567890123456789"

# 監視対象のチャンネルID
monitored_channels:
  - "345678901234567890"  # 監視したいチャンネルIDを追加
  - "456789012345678901"

# 削除時に警告メッセージを送信するか
send_warning: true

IDの取得方法

BOT IDやチャンネル IDを取得する方法

  1. Discordの「設定」→「詳細設定」→「開発者モード」を有効化
  2. BOTまたはチャンネルを右クリック
  3. 「IDをコピー」を選択
5

BOTを起動

設定が完了したらBOTを起動します

Windows

start.bat

Mac/Linux

python main.py

起動成功！

BOTが正常に起動すると、以下のようなメッセージが表示されます：

🤖 Spam Blocker でログインしました 📱 モバイルステータスで表示されています 🌐 1 サーバーに接続中

トラブルシューティング

よくある問題と解決方法

BOTが起動しない

  • config.yamlに正しいBOTトークンが設定されているか確認
  • 必要なライブラリがインストールされているか確認
  • Pythonのバージョンが3.8以上か確認

BOTが投稿を削除しない

  • BOTに「メッセージを管理」権限があるか確認
  • monitored_channelsに正しいチャンネルIDが設定されているか確認
  • Discordの開発者モードが有効になっているか確認

コマンドが動作しない

  • コマンドを実行するユーザーに管理者権限があるか確認
  • コマンドの前に「!!!」が付いているか確認
  • BOTがオンライン状態か確認

次のステップ

BOTが起動したら、以下の設定を行いましょう

許可BOTを追加

!!!add_bot コマンドで許可したいBOTを追加

監視チャンネルを設定

!!!add_channel コマンドで監視したいチャンネルを追加
